3.3 Remboursements
En cas d'annulation, les remboursements suivent les politiques choisies par les propriétaires : - Strictes : - Remboursement intégral dans les 48H suivant la réservation (si date de d'arrivée est dans 14 jours ou plus). - Remboursement à 50% pour les annulations d’au moins 7 jours avant l'arrivée. - Aucun remboursement pour annulations dans les 7 jours précédant l'arrivée. - Flexibles : - Remboursement intégral jusqu'à 3 jours avant l'arrivée. - Non remboursables : - Option avec 10% de réduction, mais sans remboursement en cas d'annulation. Une facture d'avoir est éditée en cas de remboursement.
